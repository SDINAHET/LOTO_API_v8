Suivre ses tickets Loto devient simple, visuel et précis.

Loto Tracker est mon projet RNCP6 : une application web qui permet de créer un compte, enregistrer ses tickets, comparer automatiquement ses numéros avec les tirages officiels, puis calculer les gains du tirage principal et du second tirage.

Calcul des gains : logique métier

Le projet ne se contente pas d’afficher les numéros. Il reproduit la logique métier d’un vrai ticket : nombre de bons numéros, présence ou non du numéro Chance, rang du tirage principal, puis ajout éventuel du gain du second tirage.

Tirage principal

Le ticket est comparé aux champs MongoDB boule1 à boule5 et numeroChance. Selon le résultat, le script détermine un rang de 1 à 9 puis lit le champ rapportDuRangX.

Second tirage

Les mêmes 5 numéros sont comparés aux champs boule1SecondTirage à boule5SecondTirage. Si 2, 3, 4 ou 5 numéros correspondent, le script ajoute le champ rapportDuRangXSecondTirage.

Ticket utilisateur → 5 numéros + 1 chance
Tirage principal → boule1..boule5 + numeroChance + rapportDuRang1..9
Second tirage → boule1SecondTirage..boule5SecondTirage + rapportDuRang1SecondTirage..4
Gain total → gain principal + gain second tirage
